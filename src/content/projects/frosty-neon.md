Personal website and digital portfolio. Astro static site with React islands, Tailwind CSS design tokens, and framer-motion interactions. Features a glass-panel design system with dark/light theme, neon accent colors (cyan / violet / amber), Signal blog with frequency/noise channels, academic paper showcase with PDF preview, and a macOS-style dock navigation. Fully responsive, deployed on GitHub Pages.
